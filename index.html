Viewport

we put the following in the head section in order to make our webpage adapt for different devices:
? meta name=" viewport" content="width=device-width,initial-scale=1.0" ?


Media Queries

we use media queries to control the styling of the page under certian conditions
@media (min-width:600px) {
body {
background-color: rgb(229, 84, 84);
}
}
@media (max-width: 599px) {
body {
background-color: yellowgreen;
}
}
